\section*{Key publications and documents}

south-western Australia following complete crown scorch. Fire Ecology
\textbf{11} (3) 95-107. \emph{doi: 10.4996/fireecology.1103095}

\section*{Knowledge Transfer}
Findings from this project have informed recovery actions following
large bushfires in the Warren Region including the 2012 Babbington and
2015 O'Sullivan fires.

Knowledge gained from this project has been incorporated into Fire
Management Guideline E2 Tingle forest.
